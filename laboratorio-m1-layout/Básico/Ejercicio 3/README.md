# Ejercicio 3

Para poder visualizar el ejercicio va a ser necesario ejecutar los siguientes pasos:

- Abrir un terminal en la carpeta ./Ejercicio 3/
- Ejecutar comando: npm i
- Ejecutar comando: npm run dev
- Abrir una pestaña del navegador con la url [http://localhost:5173](http://localhost:5173)

La pantalla responde a diferentes resoluciones. Para comprobarlo, cambie la anchura de la ventana y a los 550px de anchura, la barra de navegación aparecerá como columna.
